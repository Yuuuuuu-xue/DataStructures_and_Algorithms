## LeetCode Problem
Easy 217

## Description
Given an integer array `nums`, return `true` if any value appears **at least twice** in the array, and return `false` if every element is distinct.

## Example
```
Input: nums = [1, 2, 3, 1]
Output: true

Input: nums = [1, 2, 3, 4],
Output: false
```

## Basic Idea
Return size of set(input_array) != size of input_arry.
