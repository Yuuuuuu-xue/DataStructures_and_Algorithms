## LeetCode Problem
Easy 94

## Description
Given the `root` of a binary tree, return the inorder traversal of its nodes' values.

## Example
```
Input:
  1
   \
    2
   /
  3
Output: [1, 3, 2]
```

## Key Idea
Simply traverse left, root, right

## Complexity Analysis
- Runtime Complexity: O(n)
- Space Complexity: O(n)
