## Problem
Given the `root` of a binary tree, return the postorder traversal of its nodes' values.

## Example
```
Input:
    1
     \ 
      2
     /
    3
Output: [3, 2, 1]
```

## Key Idea
Post order traversal: left - right - root

## Complexity Analysis
- Runtime Complexity: O(n)
- Space Complexity: O(1)
