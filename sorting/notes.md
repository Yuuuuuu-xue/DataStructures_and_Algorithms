## Sorting
O(n^2) sorting algorithms:
- selection sort
- insertion sort
- bubble sort

O(n lg n) sorting algorithms:
- [Merge Sort](merge_sort/description.md)
- [Quick Sort](quick_sort/description.md), worst case O(n^2) but average case O(n lg n).
